## 简介
LibreSprite 是一个用于创建与制作像素动画的自由开源程序。
* 实时动画预览。
* 洋葱皮（残影）功能。
* 可同时编辑多个精灵。
* 内置调色板或自定义调色板。
* 精灵由图层与帧共同组成。
* 平铺绘制模式，适合绘制图案与纹理。
* 像素级精确工具，如填充轮廓、多边形、明暗模式等。
* 支持多种精灵与动画文件格式。

## 历史
LibreSprite 起源于 [Aseprite](https://www.aseprite.org) 的一个分支，原作者为 [David Capello](https://github.com/dacap)。Aseprite 曾以 GNU GPL v2 许可证分发，但在 [2016 年 8 月 26 日](https://github.com/aseprite/aseprite/commit/5ecc356a41c8e29977f8608d8826489d24f5fa6c)改为专有许可证。

该分支基于 GPL v2 许可的[最后一次提交](https://github.com/aseprite/aseprite/commit/03be4aa23db465219962f4c62410f628e7392545)创建，目前已独立于 Aseprite 进行开发。
